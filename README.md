# GreenTech Solutions Website

## Project Description

This project is a responsive and accessible website created for GreenTech Solutions, a company focused on environmental sustainability and eco-friendly technology. The website was developed using HTML5, CSS3, and JavaScript to provide information about the company’s mission, products, services, sustainability resources, and contact information.

The purpose of the website is to educate users about sustainable technology solutions while providing an easy to navigate and visually appealing experience for visitors.
